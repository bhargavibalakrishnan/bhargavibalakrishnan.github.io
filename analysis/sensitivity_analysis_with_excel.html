Product Price Sensitivity Analysis using MS Excel

iPhone 15: How Much Should Apple Charge for a Profitable Quarter?

Problem definition (Scope)

iPhone sales growth has declined in recent years, while its services business has grown exponentially. Sales are highest in Q1 (October to January) and decline in subsequent quarters. Q1 2023 ended with $65 billion in sales. I want to conduct a price sensitivity analysis for the iPhone15 to identify the optimal price point for profitability in Q1 2024.

Assumptions

1. Goal is to have a 2% increase in revenue from last year (2023), equivalent to $67 billion

2. The 128 GB prominent model is a good indicator of all the other storages, with a Pareto distribution of 80/20

3. The unit price of the iPhone will increase by $50 this year

Data extraction and cleaning steps

1. Install Beautiful Soup, Selenium using pip command

2. Import the necessary libraries - Pandas, BeautifulSoup, Requests, Webdriver using import command

3. Initialize the Selenium driver and use driver.get() command to navigate to the Brady's LOTO product page

driver.get('https://www.bradyid.com/lockout-tagout/tags')

4. Gather all the HTML element that contains the data using Selenium commands - driver.find_element, content.get_attribute() and BeautifulSoup()

5. Scrape the product name and price using BeautifulSoup command - find_all()

6. Clean the extracted data using strip(), split(), find(), contains() and slice() commands

6. Create a Pandas pd.DataFrame() to store the scraped data

7. Save the DataFrame to a file using to_csv() command
